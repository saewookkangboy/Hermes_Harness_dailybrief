# LinkedIn Step 2 — Strategy (M3)

**입력:** `{date}_linkedin-analysis.md`, brief 콘텐츠 캘린더
**선택:** 전략 프로필 (목표·관객·주당 게시·톤)

## 목적

분석을 **단일 포스트 전략**으로 압축합니다.

## 출력

`content/packages/{date}_linkedin-strategy.md`

```markdown
# LinkedIn 글 전략 — {date}

## 목표
- 인지 / 신뢰 / 댓글 (1개 선택)

## Hook 설계 (2줄)
**Line 1:** (호기심·숫자·반전)
**Line 2:** (가치 약속)

## 본문 아키텍처
- Context: 2–3문장 (why now)
- Insight: → 불릿 3–5
- Actionable: 1문장
- CTA: 댓글 질문

## 링크 전략
- URL: 첫 댓글 (본문 없음)

## 해시태그 후보
# ... (3–5)

## 근거 인사이트
- brief #N, search_context URL
```

## 프레임워크 (strategy-prompts)

- AIDA: Attention=Hook, Interest=Context, Desire=Insight, Action=CTA
- 콘셉트 정교화: "한 문장 포지션"
